## Install

We recommend to scaffold your first project with the [nuxt-storyblok-starter](https://github.com/regenrek/nuxt-storyblok-starter) template we have built.

<alert>Nujek requires Node.js >=12.0.0</alert>

```bash
# install degit before if its missing -> npm install -g degit
npx degit regenrek/nuxt-storyblok-starter my-website
```

## Setup Storyblok API

Since nujek is a blok builder for Storyblok you will need to connect your Storyblok project with an API Key. 

In your nujek root dir copy `.env.example` to `.env` and fill in your
API key in the `SB_CLIENT_ACCESS_TOKEN` env var. If you need detailed
instructions see [Setup Storyblok](/resources/setup-storyblok) section.
